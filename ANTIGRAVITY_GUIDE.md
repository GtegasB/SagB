# Guia de Início: Antigravity & Você 🚀

Bem-vindo! Eu sou seu agente de IA. Pense em mim como um programador sênior sentado ao seu lado, pronto para escrever código enquanto você dirige o projeto.

Já que você está começando na programação, aqui vai o segredo: **Você não precisa saber escrever o código, você precisa saber dizer O QUE você quer.**

## 1. O Básico: Como conversar comigo?
Evite tentar "adivinhar" o código. Fale sobre o **resultado** que você quer na tela.

| ❌ Evite (Muito Técnico/Confuso) | ✅ Prefira (Orientado a Objetivo) |
| :--- | :--- |
| "Mude a div x para display flex" | "Quero que os botões fiquem lado a lado." |
| "Crie um loop for no array" | "Liste todos os nomes dos Agentes na tela." |
| "O código quebrou" | "Apareceu um erro vermelho dizendo 'undefined'. Pode corrigir?" |

## 2. Minhas "Ferramentas" (O que aparece na sua tela)
Quando o trabalho é complexo, eu não saio fazendo de qualquer jeito. Eu uso "Artifacts" (arquivos de apoio) para me organizar. Eles aparecem na nossa conversa:

### 📋 `task.md` (Lista de Tarefas)
É o meu "to-do list". Eu quebro seu pedido em passos pequenos.
- `[ ]` O que falta fazer.
- `[/]` O que estou fazendo agora.
- `[x]` O que já terminei.

### 📝 `implementation_plan.md` (Plano de Voo)
Antes de alterar códigos sensíveis, eu escrevo aqui **como** vou resolver o problema.
**Dica de Mestre:** Sempre leia o topo desse arquivo. Se eu entendi errado seu pedido, é a hora de me avisar!

## 3. O Ciclo de Ouro (Workflow)
Programação com IA funciona em ciclos. Não tente fazer o app inteiro de uma vez.
1. **Peça**: Uma funcionalidade por vez. Ex: "Crie o botão de Login".
2. **Eu Planejo**: Eu analiso os arquivos e crio o plano.
3. **Eu Executo**: Eu escrevo o código nos arquivos (`.tsx`, `.css`, etc).
4. **Você Testa**: Você olha o site. Ficou bom? Pede a próxima coisa. Deu erro? Me avisa.

## 4. Exercício Rápido
Para testar, que tal fazermos uma mudança visual simples agora?
Volte para o chat e me peça:
> *"Mude a cor de fundo do menu lateral para um tom mais escuro."* 
(Ou qualquer outra cor que você prefira).

## 5. E se eu não gostar? 😱
Sem pânico! Código não é pedra, é massinha de modelar.
- **Opção A:** Diga *"Desfaça a última alteração"*. Eu volto tudo como estava antes.
- **Opção B:** Diga *"Não gostei, mude para X"*. Nós ajustamos até ficar perfeito.
- **Opção C:** Diga *"Tente de outra forma"*. Eu busco uma solução alternativa.

**Lembre-se:** Eu tenho acesso a todos os seus arquivos. Se estiver com dúvida sobre como algo funciona, basta perguntar: *"Explique o que o arquivo App.tsx faz"*.
